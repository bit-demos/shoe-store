import { LightTheme } from './light-theme';

A wrapper component that uses the theme provider and passes in the dark theme. You can then just wrap any component in the `<LightTheme>` component for the dark theme to be applied.

Alternatively you can import the dark theme and use the theme provider overrides prop to add the dark theme.

The Dark Theme uses all of the default tokens and values and overrides any values that have been set in the `light-theme-token.tsx` file. Any new tokens can also be added here and used.

### Component usage

```js
<LightTheme>
  <p style={{ backgroundColor: 'var(--bg-color', color: 'var(--text-color)' }}>
    hello world!
  </p>
</LightTheme>
```

```js
<Theme.ThemeProvider overrides="lightTheme">
  <p style={{ backgroundColor: 'var(--bg-color', color: 'var(--text-color)' }}>hello world!</p>
</Theme.ThemeProvier>
```
