import { PinkTheme } from './pink-theme';
import { ThemeContextProvider } from '@learn-bit-react/base-ui.themes.theme-provider';

A wrapper component that uses the theme provider and passes in the pink theme. You can then just wrap any component in the `<PinkTheme>` component for the pink theme to be applied.

Alternatively you can import the pink theme and use the theme provider customizedTheme prop to add the pink theme.

The pink Theme uses all of the default tokens and values and overrides any values that have been set in the `pink-theme-token.tsx` file. Any new tokens can also be added here and used.

### Render hello world in pink Theme!

```js live
<PinkTheme>
  <div
    style={{
      fontFamily: 'var(--font-family-mono)',
      padding: '0 20px',
      borderStyle: 'solid',
      borderSize: 'var(--border-size-1)',
      borderColor: 'var(--border-color)',
      borderRadius: 'var(--border-radius-medium)',
      backgroundColor: 'var(--bg-color)',
      color: 'var(--primary-color)',
      fontSize: 'var(--font-lg)',
      boxShadow: 'var(--box-shadow-x-large)'
    }}
  >
    <p>hello world!</p>
  </div>
</PinkTheme>
```

### Component usage

```js
<PinkTheme>
  <p
    style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}
  >
    hello world!
  </p>
</PinkTheme>
```

```js
<ThemeContextProvider customizedTheme="PinkTheme">
  <p
    style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}
  >
    hello world!
  </p>
</ThemeContextProvider>
```
