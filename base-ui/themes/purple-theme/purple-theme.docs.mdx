import { PurpleTheme } from './purple-theme';
import { ThemeContextProvider } from '@learn-bit-react/base-ui.themes.theme-provider';

A wrapper component that uses the theme provider and passes in the purple theme. You can then just wrap any component in the `<PurpleTheme>` component for the purple theme to be applied.

Alternatively you can import the purple theme and use the theme provider overrides prop to add the purple theme.

The purple Theme uses all of the default tokens and values and customizedTheme any values that have been set in the `purple-theme-token.tsx` file. Any new tokens can also be added here and used.

### Render hello world in purple Theme!

```js live
<PurpleTheme>
  <div
    style={{
      fontFamily: 'var(--font-family-mono)',
      padding: '0 20px',
      borderStyle: 'solid',
      borderSize: 'var(--border-size-1)',
      borderColor: 'var(--border-color)',
      borderRadius: 'var(--border-radius-medium)',
      backgroundColor: 'var(--bg-color)',
      color: 'var(--primary-color)',
      fontSize: 'var(--font-lg)',
      boxShadow: 'var(--box-shadow-x-large)'
    }}
  >
    <p>hello world!</p>
  </div>
</PurpleTheme>
```

### Component usage

```js
<PurpleTheme>
  <p
    style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}
  >
    hello world!
  </p>
</PurpleTheme>
```

```js
<ThemeContextProvider customizedTheme="PurpleTheme">
  <p
    style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}
  >
    hello world!
  </p>
</ThemeContextProvider>
```
