import { PurpleTheme } from './purple-theme';

A wrapper component that uses the theme provider and passes in the purple theme. You can then just wrap any component in the `<PurpleTheme>` component for the purple theme to be applied.

Alternatively you can import the purple theme and use the theme provider overrides prop to add the purple theme.

The purple Theme uses all of the default tokens and values and overrides any values that have been set in the `purple-theme-token.tsx` file. Any new tokens can also be added here and used.

### Component usage

```js
<PurpleTheme>
  <p
    style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}
  >
    hello world!
  </p>
</PurpleTheme>
```

```js
<Theme.ThemeProvider overrides="PurpleTheme">
  <p style={{ backgroundColor: 'var(--bg-color', color: 'var(--primary-color)' }}>hello world!</p>
</Theme.ThemeProvier>
```
