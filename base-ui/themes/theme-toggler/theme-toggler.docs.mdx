import { ThemeToggler } from './theme-toggler';

Uses the [ThemeProvider](https://bit.dev/learn-bit-react/base-ui/themes/theme-provider) component to store active theme in context which is then passed to the provider. The button onclick toggles between dark and light theme which are merged with the base and customized theme given to the provider.

### Component usage

```js
<ThemeToggler />
```
