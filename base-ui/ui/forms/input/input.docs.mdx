import { Input } from './input';

## Input Component

All props from the `<input>` element are supported. See compositions to see an example of input using state.

### Renders a basic Input

```js
<Input />
```

### Renders an Input with placeholder and id

```js live
<Input placeholder="write something here" id="my-input" />
```

### Renders an Input with placeholder and label

```js live
<>
  <label htmlFor="input">Input</label>
  <Input placeholder="write something here" id="input" />
</>
```
