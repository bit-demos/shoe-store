import { CreateCartContext } from './cart-context';

Creates a generic cart context using generic Product from the entity component with functionality to add and remove products from the cart.

Context includes:

- cart: items in cart
- addProductToCart: adds products to cart
- removeProductFromCart: removes products from cart

### Using the Context

```js
const MockUpdateContextComponent = () => {
  const context = useContext(contextObject);
  const item = {
    id: '123',
    name: 'Test Product',
    price: 10,
    quantity: 1
  };

  return (
    <div>
      <button
        className="bg-blue-500 mt-2 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded p-20"
        onClick={() => context.addProductToCart({ item, quantity: 1 })}
      >
        Add to Cart
      </button>
    </div>
  );
};
```

### Using the Cart Provier

Cart Context Provider can be used to wrap components that need to access the cart.

```js
const contextObject = CreateCartContext<Product>();

<CartContextProvider context={contextObject}>
  <Component1 />
</CartContextProvider>
```
