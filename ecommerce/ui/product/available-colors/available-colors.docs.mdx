import { AvailableColors } from './available-colors';

Takes an array of availableColors and maps over it to create circles with background color of the colors in array. This can be used to show the colors available of the products.

### Renders an array of available colors

```js live
<AvailableColors availableColors={['red', 'green', 'blue']} />
```
