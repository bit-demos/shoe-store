Counter increments and decrements by 1.

### Simple Counter

Uses default values.

```js live
<Counter quantitySelected={(count) => null} />
```

### Alets everytime counter is clicked

`quantitySelected` prop is included to pass the state of the component to the parent component

```js live
<Counter
  quantitySelected={(count) => {
    alert(count);
  }}
/>
```

### Sets the min and max Values

`min` and `max` prop to change what number counter starts at and ends at.

```js live
<Counter min={4} max={10} quantitySelected={(count) => null} />
```

### Sets the increment and decrement Values

`increment` and `decrement` prop to change what number counter starts at and ends at.

```js live
<Counter
  increment={5}
  decrement={5}
  min={0}
  max={50}
  quantitySelected={(count) => null}
/>
```
