- Counter increments and decrements by a value, 1 being the default.
- Max and min values can be set.
- OnCountChange function registers the count so it can be retrieved in another component.

### Simple Counter

Uses default values.

```js live
<Counter onCountChange={(count) => null} />
```

### Sets the min and max Values

`min` and `max` prop to change what number counter starts at and ends at.

```js live
<Counter min={4} max={10} onCountChange={(count) => null} />
```

### Sets the increment and decrement Values

`increment` and `decrement` prop to change what number counter starts at and ends at.

```js live
<Counter
  increment={5}
  decrement={5}
  min={0}
  max={50}
  onCountChange={(count) => null}
/>
```

### Shows how to retrieve Count value and use in another component

`onCountChange` prop is included to pass the state of the component to the parent component

```js
const [count, setCount] = useState(1);

<Counter
  onCountChange={(count) => {
    setCount(count);
  }}
/>
<p>Count is: {count}</p>
```
