import { Rating } from './rating';
import { FigmaEmbed } from '@learn-bit-react/base-ui.figma.figma-embed';

- `stars` props is used to pass in the amount of stars to render.
- `setStars` prop is used to set the amount of stars to store in state.

### Rating of 5 stars

```js live
<Rating stars={5} />
```

### Value is in state and can be set on click

```js live
<Rating stars={5} setStars={true} />
```
