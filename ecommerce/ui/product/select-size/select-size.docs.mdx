- Uses the label and select components to form the selected size component.
- Select element accepts an array of numbers.
- OnChange function saves the selected size so it can be used in another component.

### Set the availabe Sizes and the selected size

Function for size selected so as size can be updated or saved when using in another component.

```js live
<SelectSize
  sizeSelected={(size) => {
    return size;
  }}
  availableSizes={[45, 46, 47]}
/>
```
