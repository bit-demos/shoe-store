Uses Hero and Heading component from base-ui to create a banner component with background image and text.

### Component usage

```js
<Banner
  background="url(https://images.unsplash.com/photo-1592860986140-d77ede8b7116?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=2100&q=80) no-repeat center center scroll"
  text="banner text"
/>
```

### Adds a background image and text to the banner

```js live
<Banner
  background="url(https://images.unsplash.com/photo-1592860986140-d77ede8b7116?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=2100&q=80) no-repeat center center scroll"
  text="banner text"
/>
```
