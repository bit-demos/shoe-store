A card component to render featured products. Uses Img, Heading, Currency and Button component from both base-ui scope and ecommerece scope.

### Component usage

```js
<FeaturedCard
  src="https://images.unsplash.com/photo-1542291026-7eec264c27ff?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=crop&w=2370&q=80"
  alt="image description"
  title="Product name"
  price={190}
  id="m-1"
/>
```

### Using props to customize the card

```js live

  src="https://images.unsplash.com/photo-1542291026-7eec264c27ff?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&auto=format&fit=crop&w=2370&q=80"
```
